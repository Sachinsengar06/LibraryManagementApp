Library Management Website

This website is made using Angular framework for frontend and dotnet core webapi for backend, for styling purpose Bootstrap 5 and css is used.
In this website there are 4 seeded Users which can Lent books and borrow books, initailly each user has some tokens when they borrow a book, if it is available then a token will be deducted and a token will be added to the user who has lented this book.
Each user can add their own books and borrow books lented by others. Books can only be edited by that user who has lented them.
This website has a login verification page only logged in user can access the borrowing and lenting feature,if a user is not logged in and if he tries to borrow the book he will be redirected to the login page.
